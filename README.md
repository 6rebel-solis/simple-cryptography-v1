## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/xsolisortusx/cryptor

# Install Fernet
$ pip install Fernet

# Access
$ cd cryptor

# Run this project
$ python3/python cryptor.py
```
